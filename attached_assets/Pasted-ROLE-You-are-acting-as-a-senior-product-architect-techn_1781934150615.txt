STRATEGIC CONTEXT
ReFolDec stands for Recursively Folding Codec.

It is the keystone concept above several related OKHP³ projects:
- Mermaid Theme Builder
- BPMN for Mermaid
- skillz
- PathScrib-R / Flowpilot Scribbler lineage
- OverKill-Hill public site

ReFolDec is not merely a utility, app, or file converter. It is the overarching theory and platform vocabulary for recursively folding and unfolding ideas between formats:

raw thought
→ prose
→ structured notes
→ process narratives
→ diagrams
→ code
→ documentation
→ agent instructions
→ governed reusable artifacts
→ revised thought

The core thesis:
ReFolDec is a recursive transformation grammar for moving ideas between human-readable, machine-readable, visual, procedural, and agent-executable forms.

The newest name wraps one of the oldest concepts in Jamie Hill / OKHP³ thinking: ideas can be encoded, decoded, folded, unfolded, and refolded across media without losing semantic continuity.
