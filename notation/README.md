# BPMN-for-Mermaid

This directory contains the **BPMN-for-Mermaid** component — the visual notation layer for linked process documentation within ReFolDec.

BPMN-for-Mermaid extends Mermaid's native diagram types with BPMN-aware notation, making it possible to represent linked process flows in Markdown-native documentation.

---

## Role in ReFolDec

BPMN-for-Mermaid sits on the **Visual plane** of the ReFolDec architecture:

```
Executable layer: Agent Skills, prompts, runbooks, SOPs
  ↓
Visual layer: Mermaid, BPMN-for-Mermaid, linked diagrams  ← here
  ↓
Version/publish layer: GitHub, OverKill Hill, static sites
```

It is a **component** of ReFolDec — not an independent product.
